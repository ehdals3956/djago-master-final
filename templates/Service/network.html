VPC

가상 사설 클라우드 상 리소스 배치,
연결 및 보안을 포함하여 가상
네트워킹 환경을 제어하는 사용자의
AWS 계정 전용 가상 사설 네트워크

API Gateway

API를 손쉽게 생성, 게시, 유지 관리,
모니터링 및 보안 유지할 수 있도록
하는 완전 관리형 서비스로서
컨테이너식 서비리스 워크로드 지원

Direct Connect

고객이 리전에 있는 여러 VPC를
Direct Connect를 통해 데이터
센터와 오피스 사이에서 프라이빗
네트워크 연결을 생성하는 서비스









AGA

AWS의 글로벌 네트워크 인프라를
사용하여 퍼블릭 애플리케이션의
가용성, 성능 및 보안을 개선하는데
유용한 네트워킹 서비스

Cloud Front

서버나 스토리지 등의 서비스의
컨텐츠 전송 속도를 높혀 성능,
보안 및 개발 편의를 위해 구축된
컨텐츠 전송 네트워크 서비스

Route 53

가용성과 확장성이 뛰어난
AWS DNS 웹 서비스
