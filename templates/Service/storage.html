Storage

                                   시공간 제약 없이 데이터를 저장하고 효율적으로 활용할 수 있는 저장소를 제공하는 서비스                                     



끊임없는 기술 탐구와 수준 높은 성능 보장, 서비스 특성에 최적화와
데이터 유실 걱정 없는 백업 정책을 자랑하는 스토리지 인프라 제공



Storage Services




S3

업계 최고 수준의 확장성, 데이터
가용성, 보안 및 성능을 제공하는
객체 스토리지 서비스

S3 Glacier

자주 사용되지 않는 데이터를 보관
및 백업하는데 유용하며, 확장성이
뛰어나며 빠르게 액세스가 가능한
저비용 아카이브 스토리지

Block Storage

서버에서 하드디스크처럼 사용하여
사용이 쉽고 확장 가능한
고성능 블록 스토리지









File System

서버에서 파일을 추가 및 제거할 떄
자동으로 확장/축소가 되며 관리
또는 프로비저닝이 필요없는
파일 공유 스토리지

FSx

서버 메시지 블록 프로토콜을 통해
액세스가 가능한 고도로
안정적이고 확장 가능한
완전관리형 파일 스토리지

Backup

정책을 기반으로 대규모 데이터
백업을 중앙 집중화하고 비용
효율적으로 보호할 수 있는
완전 관리형 스토리지 백업 서비스









S3 Gateway

S3와 로컬 스토리지를 연동해주는
서비스로 로컬에 있는 장비와 S3를
연동해서 메인 데이터와 접근
빈도가 높은 데이터를 나누는 서비스
