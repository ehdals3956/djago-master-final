Security

                          세계적 수준의 보안 기술을 통해 외부 위협으로부터 고객의 서비스와 데이터를 안전하게 지켜드립니다                          



고객의 정보 보호를 최우선으로 튼튼한 보안 환경을 제공, 글로벌 최고 수준의
보안 인력 운영을 더불어 안전한 클라우드 서비스 제공을 보장하는 국내외 보안 인증 획득



Security Services




WAF

웹의 가용성에 영향을 주거나
보안을 훼손하는 공격으로부터
웹 애플리케이션, API를 보호하는
웹 애플리케이션 방화벽

Firewall Manger

계정과 애플리케이션에 대한
방화벽 규칙을 중앙에서 구성하고
관리할 수 있는 보안 관리 서비스

KMS

KMS는 클라우드 서비스
사용자만의 환경을 암호화 키를
사용하여 관리할 수 있는
멀티 테넌트 관리형 서비스









Secret Manager

코드의 암호화를 포함하여 자격
증명을 프로그래밍 방식으로 보안
암호를 검색하여 안전하게 저장 및
검색할 수 있는 암호 관리 서비스

Detective

리소스에서 로그 데이터를
자동으로 수집하고, 기계 학습,
통계 분석 및 그래프 이론을 사용하여
보다 효율적인 보안 관리 서비스
