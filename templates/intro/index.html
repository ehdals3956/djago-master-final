회사소개

골드문은 기업용 IT솔루션과 하드웨어와 소프트웨어를 국내외 대기업, SMB, ISV 등
공공 및 교육기관등에 공급하고 있으며
글로벌 기업의 반도체 설계과제를 Cloud를 통해 설계, 구축, 관리 운영하고 있습니다.

Customer Centric
"고객 중심"

고객부터 자영업, 스타트업, 중소기업 나아가 대기업까지 비지니스 성공을 위한 다양한 분야의 전문가들을 위한 최적의 설루션 제공

Anyone
"누구에게나"

누구나 쉽게 사용할 수 있는 클라우드 서비스

Easy
"고객이 이용하기 쉬운"

웹 브라우저 상에서 간단한 클릭만으로 서비스 사용

Efficient
"합리적인 비용"

합리적인 비용으로 인프라와 플랫폼 이용 가능

연혁

2022

  1. - GitHub 파트너십 체결
  2. - 마이크로소프트 Security 전문 파트너
  3. - VM ware 파트너 계약
  4. - 클라우드 매니지드 사업, 기업 우수상 수상
  5. - 사옥이전 및 네이버 클라우드 플랫폼 파트너 체결




2020

  1. - Amazon Web Service 파트너 계약
  2. - 전북은행 온프레미스 -> AWS 클라우드 서비스
  3.    마이그레이션 사업 진행
  4. - 애터미 AWS 클라우드 솔루션 도입 사업 진행




2018

  1. - AWS 클라우드 디지털 서비스 선정(매니지드)
  2. - 엔터업plan 투자유치(16억)
  3. - 제조업plan 투자유치(13억)
  4. - 소매업plan 투자유치(7억)
Amazon Web Services
AWS

데이터 레이크 구축하고 amazon athena로 분석하여 분석결과를 토대로 결정과 급증하는 트래픽 대응위해 서버 다운 타임을 내리기가 필요했고 다양한 매니지드 서비스를 제공받을 수 있다.

AWS로 이동
Net3 Technology
Net3 TECH

미 클라우드 회사로 백업 IaaS, 재해 복구 그리고 aws, azure, gcp와 자체 Pvdc cloud를 포함하여 다루는 플랫폼 서비스 제공


Net3 TECH로 이동
